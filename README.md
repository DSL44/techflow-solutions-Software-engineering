# techflow-solutions-Software-engineering

Gerenciamento de tarefas baseado em metodologias ágeis

Objetivo:

Simular o desenvolvimento de um sistema de gerenciamento de tarefas usando metodologias ágeis.

Escopo:

- Criar um sistema simples de tarefas
- Permitir criar, listar e remover tarefas
- Utilizar GitHub para controle de versões e gestão ágil

Metodologia utilizada:

Kanban, utilizando o GitHub Projects.

Ferramentas utilizadas:

- Python
- Pytest
- GitHub Actions
